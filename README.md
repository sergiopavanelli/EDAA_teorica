## 🎯 Sobre o Repositório

Este repositório reúne todas as atividades desenvolvidas nas aulas teóricas da disciplina **Estrutura de Dados e Análise de Algoritmos (EDAA)**. As atividades são organizadas por aula em pastas específicas e serão adicionadas progressivamente ao longo do semestre.

O objetivo central da disciplina é desenvolver raciocínio lógico e computacional para a resolução de problemas, aplicando os principais conceitos de **estruturas de dados** (listas, pilhas, filas, árvores, grafos) e **análise de complexidade de algoritmos** (notação Big-O, análise temporal e espacial).
